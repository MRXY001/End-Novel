# 终点小说网

唔，这个是 J2EE 的课程设计……

纯 JSP，无后端框架。



## 功能

首页从起点中文网中抓取小说数据（封面、书名、作者、简介等），以及各类榜单。

简单的注册登录功能，包括滑动滑块发送短信验证（收费）。

发表作品，显示作品，审核作品（假）功能。

其余一些小细节。
